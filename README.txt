
This python script is designed to convert audio to text. You can see more examples of using vosk-api here: https://github.com/alphacep/vosk-api

1. Script works native with WAVE files only or with any audio by using FFMPEG. https://ffmpeg.org/

2. Please check WAVE_MODE and FFMPEG_DIR in config.py

3. Download model for vosk-api here: https://alphacephei.com/vosk/models

4. Please check MODEL_DIR in config.py
